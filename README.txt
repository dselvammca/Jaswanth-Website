
How to use this site

1) Upload the entire folder to your host.
2) Replace images in assets/images/:
   - Gallery: j1.jpg … j30.jpg
   - Inspirations: R1.jpeg, L1.jpeg, r2.jpeg … r6.jpeg
3) Auto-load YouTube videos:
   - Get your Channel ID (UC....) and API key (enable YouTube Data API v3).
   - Edit assets/videos.js and set API_KEY and CHANNEL_ID.
4) Contact form:
   - On PHP hosting (Hostinger), works with contact.php (uses mail()).
   - On Vercel/Netlify (no PHP), use Formspree: change action to https://formspree.io/f/YOUR_FORM_ID in contact.html.

Tips: compress large photos; keep file names exactly as referenced; customize text on each page.
